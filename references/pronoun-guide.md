# Pronoun Guide — Genre-Specific Usage

## Overview

Pronoun choice is one of the most important decisions in CN→VN novel translation. It sets the tone and genre feel.

## By Genre

### Tiên Hiệp (tienxia)
- **Male 3rd person**: hắn (narrative), y (formal/classical)
- **Female 3rd person**: nàng
- **1st person**: ta (narrative), tôi (modern dialogue)
- **2nd person**: ngươi (formal), ngươi (dialogue)
- **Tone**: Classical, spiritual. hắn/nàng is the default pair.

### Kiếm Hiệp (wuxia)
- **Male 3rd person**: y (primary), hắn (secondary)
- **Female 3rd person**: nàng
- **1st person**: ta (martial arts context), tôi (modern)
- **2nd person**: ngươi (formal)
- **Tone**: Action-oriented. y is more martial than hắn.

### Thành Thị (urban)
- **Male 3rd person**: anh (younger), hắn (older/rival)
- **Female 3rd person**: cô ấy, nàng (if romantic interest)
- **1st person**: tôi, tao (casual/angry)
- **2nd person**: bạn, mày (casual)
- **Tone**: Modern, conversational. Avoid classical pronouns.

### Lịch Sử (historical)
- **Male 3rd person**: hắn, y
- **Female 3rd person**: nàng, thị (archaic)
- **1st person**: ta, trẫm (emperor), thần (minister)
- **2nd person**: ngươi, khanh (emperor to minister)
- **Tone**: Formal, period-appropriate. Use royal pronouns for court scenes.

### GameLit (gamelit)
- **Male 3rd person**: hắn
- **Female 3rd person**: nàng
- **1st person**: ta, tôi
- **2nd person**: ngươi
- **Tone**: Technical terms can stay English (Level, HP, MP).

### Kinh Dị (horror)
- **Male 3rd person**: hắn (ominous), y (neutral)
- **Female 3rd person**: nàng, ả (sinister)
- **1st person**: ta (ghost/entity), tôi (human)
- **2nd person**: ngươi (ghost), mày (informal/scared)
- **Tone**: Dark. Pronoun choice can signal who is human vs supernatural.

### Fantasy (fantasy)
- **Male 3rd person**: hắn
- **Female 3rd person**: nàng
- **1st person**: ta, tôi
- **2nd person**: ngươi
- **Tone**: Epic. Can borrow from tienxia for magic scenes.

## Dialogue vs Narrative

| Context | Rule |
|---------|------|
| Narrative prose | Use genre default pronouns |
| Formal dialogue | Use formal pronouns (ngươi, ta) |
| Casual dialogue | Use casual pronouns (mày, tao) |
| Inner monologue | Match character personality |
| Ancient/formal speech | Use classical (y, nàng, ta) |

## Common Mistakes

1. **Pronoun leakage**: Using ngươi/ta in narrative outside dialogue
2. **Register mismatch**: Using mày/tao in a formal scene
3. **Gender confusion**: Using hắn for a female character
4. **Inconsistency**: Switching between hắn and y for same character
